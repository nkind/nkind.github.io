Marshmallow Madness

an endless runner game on Facebook Gameroom

An individual project for the web programming course at the University of Saskatchewan. All of the code and art was created independently in less than a month, so the game would be ready to face a timely publishing process.

Facebook felt like a unique platform for achieving the goal and offered significant challenges for learning about the publishing experience. In the end, Marshmallow Madness was approved and is now available publicly for players everywhere to enjoy.
