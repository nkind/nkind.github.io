Air Prairies

a flight reservation website

The project was made for the intermediate software engineering course at the University of Saskatchewan using the Agile work process. Our goal was to create a flight booking application with a focus on usability, avoiding the typical clutter that similar applications face.

As a team, we used nodejs, express, ejs and sqlite3 to create a functional database that allows for finding and booking flights. Individually, I designed the entire front-end of the application using HTML5/CSS3.
